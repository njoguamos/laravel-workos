---

## Request

To get the details of an existing user, call the `getUser` method of the `UserManagement` class and pass the user's id as an argument.

```php
use NjoguAmos\LaravelWorkOS\UserManagement;

$response = (new UserManagement())
    ->getUser(id: 'user_01J140CFC29X7PN6FF1RVXVSRJ');
```

<Accordion title="Parameters">

    <ParamField query="id" type="string" default="none" required>
        The unique ID of the user you want to retrieve.
    </ParamField>

</Accordion>

## Response

The `response` in an instance of `NjoguAmos\LaravelWorkOS\DTOs\UserDTO`.

```php
$response->object; # \NjoguAmos\LaravelWorkOS\Enums\WorkOSObject instance
$response->object->name; # USER
$response->object->value; # user
$response->id; # user_01J140CFC29X7PN6FF1RVXVSRJ
$response->email; # brian@example.com
$response->email_verified; # true
$response->created_at; # Carbon\CarbonImmutable instance
$response->updated_at; # Carbon\CarbonImmutable instance
$response->first_name; # Brian
$response->last_name; # Kariuki
$response->profile_picture_url; # https://workoscdn.com/images/v1/123abc
```

If you prefer array, you can call  `$response->array()` .

```php
[
    "object" => "user"
    "id" => "user_01J140CFC29X7PN6FF1RVXVSRJ"
    "email" => "brian@example.com"
    "email_verified" => true
    "created_at" => "2024-06-29T19:07:33.155Z"
    "updated_at" => "2024-06-29T19:07:33.155Z"
    "first_name" => "Brian"
    "last_name" => "Kariuki"
    "profile_picture_url" => "https://workoscdn.com/images/v1/123abc"
]
```
