title: 'Get a user'
description: 'Get the details of an existing user.'
---

## Request

To get the details of an existing user, call the `getUser` method of the `UserManagement` class and pass the user's id as an argument.

<CodeGroup>

```php New Class
use NjoguAmos\LaravelWorkOS\UserManagement;

$response = (new UserManagement())->getUser(id: 'user_01J140CFC29X7PN6FF1RVXVSRJ');
```

```php Facade
use NjoguAmos\LaravelWorkOS\Facades\UserManagement;

$response = UserManagement::getUser(id: 'user_01J140CFC29X7PN6FF1RVXVSRJ');
```
 </CodeGroup>

<br/>

<Accordion title="Parameters">

    <ParamField query="id" type="string" default="none" required>
        The unique ID of the user you want to retrieve.
    </ParamField>

</Accordion>

## Response

The default response in an instance of `NjoguAmos\LaravelWorkOS\DTOs\UserDTO`, which can also be converted to an array `$response->array()` or to json `$response->json()`
